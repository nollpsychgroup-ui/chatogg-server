Step 2 — Add to iPhone Safari

  1. Tap Copy Bookmarklet Code above.
  2. In Safari, go to any page (e.g. apple.com).
  3. Tap the Share button (□↑) → Add Bookmark → Save.
  4. Open Bookmarks (open-book icon in toolbar) → Edit.
  5. Tap the bookmark you just saved.
  6. Select all text in the URL field and paste the code you copied.
  7. Tap Done. You're ready!

Step 3 — Use It

  1. Open remotedesktop.google.com in Safari and connect to your computer.
  2. Tap the address bar, type the first few letters of your bookmark name, then tap it — or open Bookmarks and tap it directly.
  3. The RDP Assist panel floats over the remote desktop screen.
  4. Drag the panel by its title bar to reposition it anywhere.
  5. Tap — to minimise it to a thin strip when you need more screen space.
  6. Tap the bookmark again to dismiss the panel.

What's in the Panel

Modifier Keys
Sticky Ctrl / Alt / Shift / Win — tap to lock on, fires with the next key press
🖱️
Right-Click
Arm right-click mode then tap anywhere on the desktop to send a right-click
⚡
Shortcuts
Copy, Paste, Undo, Redo, Save, Find, New Tab & more in one tap

Special Keys
Esc, Tab, Enter, Delete, Home, End, PgUp / PgDn, arrow keys
